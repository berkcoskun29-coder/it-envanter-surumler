# IT Envanter ve Zimmet Takip Sistemi — Sürümler

Bu depo yalnızca uygulamanın **çalıştırılabilir sürümlerini** (Releases) ve
güncelleme bilgisini (`surum.json`) barındırır. Kaynak kod burada değildir.

- En güncel sürüm: **Releases** sekmesinden `IT-Envanter.exe`.
- `surum.json`: uygulamanın açılışta okuduğu son sürüm bilgisi (otomatik güncelleme kontrolü).
